## 📂 OPERATIONAL ARCHITECTURE [BULLETPROOF LAYOUT]
Engineered for modularity. Each feature is a self-contained unit, preventing spaghetti-dependency failure.

```plaintext
src/
├── app/             # Command & Control (Entry points/Routing)
├── assets/          # Intelligence Assets (Optimized Media)
├── components/      # Global Support UI (Buttons, Brand-Logo)
├── features/

├── lib/             # Third-party Comms (Analytics, Speed-Insights)
└── utils/           # Tactical Helpers (Logic & Formatters)
```
